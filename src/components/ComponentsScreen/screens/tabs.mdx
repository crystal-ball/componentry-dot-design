## A++ A11y

- The list of tabs has a role `tablist` and should be passed an `aria-label` describing
  the tabs.
- Each tab inside the `tabslist` container should be a button with the role `tab`. Using
  buttons simplifies keyboard accessibility.
- Each tab should also have an `aria-selected`, `aria-controls`, `id` and `tabindex`
  property.
- Each content panel of the tab should have a `tabpanel` role.
- Each content panel should also have an id, a tabindex, an aria-labelledby, and a hidden
  property.

CONTAINER: `div` with role `tablist` TAB: `button` with role `tab` CONTENT: `div` with
role `tabpanel`
