<Text variant='lead'>Display menus, lists and more in a toggleable container.</Text>

```javascript
<Dropdown>
  <Dropdown.Trigger>Dropdown</Dropdown>
  <Dropdown.Content>
    <Dropdown.Item>Item 1</Dropdown.Item>
    <Dropdown.Item>Item 2</Dropdown.Item>
  </Dropdown.Content>
</Dropdown>
```

- Drodpowns don't include menu or list settings by default because they may include
  headers, dividers, inputs, etc.

## Recipes

### Menus

Menus are a stricter subset of dropdowns that should only include menu options. You can
create a menu by adding the `menu` and `menuitem` roles to the dropdown content.

<Dropdown>
  <Dropdown.Trigger>Update</Dropdown.Trigger>
  <Dropdown.Content role='menu'>
    <Dropdown.Item role='menuitem'>Save</Dropdown.Item>
    <Dropdown.Item role='menuitem'>Print</Dropdown.Item>
    <Dropdown.Item role='menuitem'>Delete</Dropdown.Item>
  </Dropdown.Content>
</Dropdown>
