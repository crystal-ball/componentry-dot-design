# Componentry Design System

_Important design system concepts_

1. The spacing and typography values are all set in rems. This allows users to customize
   their browser font sizes and we will respect that.

# Setting up responsive type/spacing

1. Select the theme font(s) first, different fonts look better at different sizes.
1. Assume that we're working with a base html font size of `16px`, and that we're going to
   setup an 8px grid system.
1. In small screens we want to set the body font size to `16px`. We use a css variable.
