# Typography

The typography values in the theme are used to create the typography utility styles. (The
Text component styles are defined seperately).

```scss
$theme: (
  typography: (
    // font-family-* classes
      font-families: (/* default: primary, monospace */),
    // font-size-* classes
      fonts-sizes: (/* sm, md, lg, body, small, code */),
    // font-weight-* classes
      font-weights: (/* light, normal, bold */),
    // letter-spacing-* classes
      letter-spacings: (),
    // line-height-* classes
      line-heights: (/* single, body */),
    // text-shadow-* classes
      shadows: (),
  ),
);
```
