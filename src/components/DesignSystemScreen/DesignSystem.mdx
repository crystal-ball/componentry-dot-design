# Design system

The Componentry design system can be customized using SASS theme variable overrides as
well as creating custom variants.

## Customizing SASS variables

Design system theme variables are defined in the Componentry [`src/styles/_variables.scss`][variables] file.
Override any of these values before importing componentry to customize the generated styles.
